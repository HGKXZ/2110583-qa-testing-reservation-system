---

## 🧩 Project Overview

This repository contains a **comprehensive QA testing project** for a:

> **Restaurant Table Reservation System (RTS)**

The system allows customers to reserve tables online while enabling staff and administrators to manage reservations, tables, and restaurant operations.

This project focuses on applying **end-to-end software testing practices** based on a formal **software requirements specification (SRS)**.

📌 System specification:
[Reservation System SRS](requirements/reservation-system.pdf)

---

## 🎯 Objectives

* Perform **requirement analysis** from SRS
* Design structured **test cases and coverage**
* Validate system behavior through:

  * Functional Testing
  * API Testing
  * UI Automation
  * Performance Testing
  * Security Testing
* Simulate a **real-world QA team workflow**
